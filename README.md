# 勤怠管理アプリ

出勤・退勤時間を記録し、勤務時間を確認できるシンプルな勤怠管理アプリです。

## URL

https://attendance-app-lyart-kappa.vercel.app/

## 使用技術

- React
- Vite
- JavaScript
- CSS
- Vercel

## 主な機能

- 出勤時間の入力
- 退勤時間の入力
- 勤務時間の表示
- スマートフォン対応
- デモ機能

## 制作目的

Reactの状態管理やコンポーネント設計の学習を目的に制作しました。  
毎日使う勤怠入力を想定し、できるだけシンプルに操作できるUIを意識しています。

## 工夫した点

- 入力項目を絞り、迷わず操作できる画面構成にしました。
- スマートフォンでも使いやすいように、ボタンや入力欄の見やすさを意識しました。
- 修正とデプロイを繰り返しながら、動作確認と改善を行いました。
- まずは体験して頂くためにデモ機能を実装しました

## 今後の改善予定

- 入力値のバリデーション強化
- 部の画面でテキスト表示が更新されない不具合の修正
- 勤務履歴の保存
- 月別の勤怠記録の表示
- 月ごとの勤務時間集計
- UIデザインの改善
- レスポンシブの見直し
- コードの整理・コンポーネント分割

## 学習メモ

制作を通して、Reactでの状態管理、フォーム入力、デプロイまでの流れを学びました。  
今後は実際の利用を想定し、データ保存や集計機能を追加していく予定です。
